# 逗子グルメ 定期更新タスク

逗子駅（JR横須賀線／京急 逗子・葉山）徒歩圏を中心とした海街グルメサイトを、
毎回ちがう「視点（angle）」で少しずつ充実させる定期運用。馬車道版と同じ流れ。

## 対象ファイル（作業フォルダ直下）
- `data/restaurants.json` … 正本（これだけを編集）
- `template.html` … 雛形（通常は触らない）
- `build_site.py` … 生成スクリプト
- `index.html` … 生成物（直接編集しない）

## 毎回の手順
1. **視点を1つ選ぶ**（下の「視点プール」から、前回までと重複しないものを `meta.updates` を見て選択）。
2. **WebSearch / web_fetch で調査**。食べログ・ヒトサラ・Retty・くふうトリップ・逗子旅(zushitrip.com)等の口コミを参照。巨大ページの丸ごと取得や重複取得は避け、必要箇所のみ。
3. **3〜6件を新規追加**（既存 `id` と重複しないこと。閉店店は載せない）。
   - `distance_min` は必ず逗子駅からの徒歩分。基準＝逗子駅 lat35.2956, lng139.5797、徒歩圏は半径約700m目安。海岸方面など徒歩圏を超える定番店は徒歩分を概算で。
   - `lat`/`lng` は必須。住所（町丁）から概算座標を付与。
   - 点数・距離・営業情報は出典に基づく。不確かな数値は `null`（捏造しない）。
   - `scene_tags` に視点に対応するタグを必ず含める。
4. **meta を更新**：`meta.updates` に `{date: 今日, angle: 選んだ視点, note: 概要}` を追加し、`generated_at` を今日に。
5. **Python で書き込み**：読み込み→追記→`json.dump(..., ensure_ascii=False, indent=2)`→書き戻し→`json.load` で妥当性確認。
6. **再生成**：`python3 build_site.py`。先頭 `<!DOCTYPE`・末尾 `</html>`・件数・`__DATA__` が残っていないことを確認。
7. **デプロイはしない**。ローカル生成まで。公開はユーザーが「デプロイして」と言ったときだけ。

## 視点プール（海街・逗子ならでは）
逗子は海街なので、定番ジャンルに加えて海らしい視点が回しやすい。

- 海鮮・しらす（小坪漁港の地魚、釜揚げ／生しらす丼）
- 海岸テラス・サンセット（逗子海岸・逗子マリーナのオーシャンビュー）
- 海街さんぽ（テイクアウト＋ピクニック、ベーカリー、ジェラート）
- 寿司・天ぷら（特別な日／大人の隠れ家）
- 朝カフェ・モーニング（海前のカフェ）
- 逗子・葉山駅（京急）側＝桜山・新宿エリア
- 久木・池子の住宅街グルメ（隠れ家・落ち着く）
- スイーツ・かき氷（百名店、手土産）
- 雨の日・通し営業（駅近で使いやすい）
- 子連れ・テラス席（家族向け）

> 既出の視点は `meta.updates` の `angle` で確認できる。毎回ちがう切り口を選ぶこと。

## restaurants.json のスキーマ（再掲）
```json
{ "meta": { "generated_at":"YYYY-MM-DD", "updates":[ {"date":"","angle":"","note":""} ] },
  "restaurants":[ {
    "id":"", "name":"", "genre":"", "distance_min":0, "rating":0.0_or_null,
    "rating_source":"", "reviews":0_or_null, "price_dinner":"", "price_lunch":"",
    "closed":null_or_str, "address":"", "lat":0.0, "lng":0.0, "tabelog_url":"_or_null",
    "awards":[], "summary":"", "signature":"", "review":"", "scene_tags":[] } ] }
```

## トークン最小化
精度を落とさない範囲で重複取得・巨大ページ丸取りを避け、最終報告は簡潔に
（選んだ視点・追加した店舗名・総件数・未デプロイの明記のみ）。

## スケジュール設定の例
「毎週月曜の朝9時に逗子グルメサイトをあたらしい視点で更新して」のように依頼すれば、
このスキルの手順で定期実行できる（cron: `0 9 * * 1`）。
